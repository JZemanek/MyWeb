    O Javě

    Java je programovací jazyk vyvinutý společností Sun Microsystems. Jedná se o jeden z nejpopulárnějších a nejuniverzálnějších programovacích jazyků.

    Je open-source, platformově nezávislý, a je implementován prostřednictvím Java Virtual Machine (JVM). Díky tomu je vhodný pro vývoj webových aplikací, Android aplikací, a dalších.
